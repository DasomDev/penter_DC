# 펫트너(펫시터 플랫폼)

### 프로젝트
+ 사용자 위치를 기반으로 내 주변의 펫시터 혹은 펫시팅이 필요한 반려동물 찾을 수 있는 서비스입니다
+ Spring Framework 을 사용하여 웹서비스를 제작했습니다
+ 사용자가 보호자 / 펫시터 두가지 경우의 수를 모두 이용 할 수 있도록 데이터 테이블을 설계하였습니다
